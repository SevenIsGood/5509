BLE Equalizer Help

Upon succesful connection of BOOSTC5545ULP, C5545 Boost App displays Equailizer screen with all the five equalizer data along with the music play status. User can set any equalizer data by dragging equalizer

Music play status(Play/Pause/Stop) shall change based on the BOOSTC5545ULP music status.
