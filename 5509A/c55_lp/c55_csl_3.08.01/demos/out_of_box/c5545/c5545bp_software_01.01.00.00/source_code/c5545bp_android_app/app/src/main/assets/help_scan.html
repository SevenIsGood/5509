BLE Scanner Help

Provided that the Bluetooth Adapter is available/turned ON and BLE is supported on the device (the application will request to turn the bluetooth on if Bluetooth is turned off at start-up), the C5545 Boost App will start device scanning on click of Scan button. Scanning will stop when any of the following conditions are met:

  • Stop button tapped
  • A device in the list tapped
  • Fifteen seconds elapsed since scanning started

Scanning can be initiated any time by tapping the Scan button. If the scan succeeds, application lists the scanned C5545 devices with the "Connect" button. User can click on the "Connect" button to establish connection with the desired BLE device.If connection is successful, BLE Equalizer screen will be launched else error message is displayed.

The options menu gives direct access to the following :

  • The Bluetooth icon to turn ON Bluetooth.
  • C5545 Boost Home : To launch C5545 Boost web page.
  • TI's e2e forum : To launch TI's e2e forum web page.
  • About : Displays the aplication version and android version of the mobile.
  • Exit : Exit from the application.
