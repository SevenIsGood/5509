int gpio_input_int_test ( void  )

Function to test the functionality of GPIO as input pin.

This function configures GPIO pin 11 as input pin and pin 4 as output pin A data value is written to pin4 and is read from pin11. Both the values are compared and the result is returned to the main function.

Connect GPIO 4 and GPIO11 on the C5515 EVM with a jumper, Pin 1 of J22 is GPIO 4 and Pin 2 of J22 is GPIO11
