Detailed Description

DMA interface definitions.

This file contains the data types and symbolic definitions that are needed by the application to utilize the services of the DMA device driver on DA225 board.

(C) Copyright 2006, Texas Instruments, Inc

Author
PR Mistral
Version
0.1 Initial implementation
