CSL_Status CSL_checkNandType ( CSL_EmifHandle  hEmif)

Checks the type of NAND flash.

This function reads the NAND flash device ID to decide whether it is big block or small block NAND. Type of the NAND device will be updated to the global variable 'gNandType'. gNandType - 0; Small Block Flash gNandType - 1; Big Block Flash
