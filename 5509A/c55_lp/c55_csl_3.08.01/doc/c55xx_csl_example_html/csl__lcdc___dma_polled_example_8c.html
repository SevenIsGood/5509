CSL_Status CSL_lcdcDmaPolledTest ( void  )

LCD DMA polled test function.

This function calls different CSL LCD APIs to configure LCD-DMA and to fill the LCD panel with Blue and Red color. DMA transfer completion is verified by polling on the LCD status register This function fills upper half of the LCD with Blue color and lower half with Red color.
