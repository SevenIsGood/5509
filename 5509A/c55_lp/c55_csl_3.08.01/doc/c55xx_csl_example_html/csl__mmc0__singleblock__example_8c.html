Uint16 computeClkRate ( void  )

Function to calculate the memory clock rate.

This function computes the memory clock rate value depending on the CPU frequency. This value is used as clock divider value for calling the API MMC_sendOpCond(). Value of the clock rate computed by this function will change depending on the system clock value and MMC maximum clock value defined by macro 'CSL_MMC_CLOCK_MAX_KHZ'. Minimum clock rate value returned by this function is 0 and maximum clock rate value returned by this function is 255. Clock derived using the clock rate returned by this API will be the nearest value to 'CSL_MMC_CLOCK_MAX_KHZ'.

Parameters
none
Returns
