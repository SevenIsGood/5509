CSL_Status MCBSP_close ( CSL_McbspHandle  hMcbsp)

============================================================================


MCBSP_close

Description


Close the handle to MCBSP and stops the clock to MCBSP

Arguments 

        None

Return Value CSL_Status

  • CSL_SOK - Returned for success 
                       CSL_ESYS_BADHANDLE - Bad handle passed

Pre Condition


MCBSP_open should be called successfully.


None

Post Condition

Example 

        Status = MCBSP_close(hMcbsp);

        if( Status != CSL_SOK )

        {

         printf("MCBSP_close is failed\r\n");

        }
