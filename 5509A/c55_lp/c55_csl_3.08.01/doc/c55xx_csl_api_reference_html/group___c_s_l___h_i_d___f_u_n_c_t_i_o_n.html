============================================================================
HID_eventHandler

Description
This function handles all HID data transactions

Arguments 

        pContext         Pointer to the USBContext structure
        hHidClass        Handle to the HID Class Obj
        usbSetUp         Structure to hold the USB Set up Values that were
                         used to configure the USB End Point

Return Value - Status of the processing (CSL_Status)

  • CSL_SOK (0) Success
  • CSL_ESYS_FAIL (-1) Generic failure
  • CSL_ESYS_BADHANDLE (-5) Handle passed to CSL was invalid
  • CSL_ESYS_INVPARAMS (-6) Invalid parameters

Pre Condition
HID module should be opened using HID_Open function

Post Condition
Returns request processing status

Modifies
None

        CSL_HidClassStruct      hidClassStruct;
