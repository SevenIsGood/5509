CSL_Status UHPI_Close ( CSL_UhpiHandle  hUhpi)

============================================================================
UHPI_Close

Description
Close the handle to UHPI and stops the clock to UHPI

Arguments 

        None

Return Value CSL_Status

  • CSL_SOK - Returned for success CSL_ESYS_BADHANDLE - Bad handle passed

Pre Condition
UHPI_open should be called successfully.
None

Post Condition

Example 

        Status = UHPI_Close(hUhpi);
        if( Status != CSL_SOK )
        {
         printf("UHPI_Close is failed\r\n");
        }
